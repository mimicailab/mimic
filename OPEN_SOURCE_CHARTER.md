# Open Source Charter

This charter defines Mimic's binding commitments to the open-source community. These are not aspirations — they are promises.

## Our Commitments

### 1. The CLI, adapter SDK, and all current open-source adapters will never move to a proprietary license.

Every package currently published under Apache 2.0 stays Apache 2.0. No relicensing, no bait-and-switch, no "we need to make a difficult business decision" blog post three years from now. If it's Apache 2.0 today, it's Apache 2.0 forever.

### 2. Features that are free today will never move to paid tiers.

The free tier is not a trial. It is a permanent, fully functional product. We will add new features to paid tiers, but we will never take existing free features and put them behind a paywall.

### 3. Community-contributed adapters remain Apache 2.0 forever.

If you contribute an adapter, it stays open source regardless of any future decisions by Mimic Data Ltd. Your contribution, your license, your terms. We cannot and will not change this.

### 4. The adapter SDK interface is stable.

Breaking changes to the adapter SDK require a public RFC, a 6-month deprecation period, and a migration guide. We will not break community adapters with surprise API changes.

### 5. If Mimic Data Ltd ceases to operate, the open-source components continue.

In the event that Mimic Data Ltd is dissolved, acquired, or otherwise ceases active development, we will transfer stewardship of all Apache 2.0 components to a community-governed foundation or designate community maintainers with full publishing rights.

## Why This Matters

Open-core is a trust relationship. You invest your time building adapters, integrating Mimic into your workflows, and recommending it to your team. That investment only makes sense if you can trust that the open-source components will remain open source.

This charter exists to make that trust concrete. It is published in the repository, referenced in the README, and included in every adapter PR template as a reminder of what we've committed to.

## Enforcement

This charter is a public commitment. If Mimic Data Ltd violates any of these commitments, the community has full right to fork the Apache 2.0 components and continue development independently. The Apache 2.0 license guarantees this right — this charter simply clarifies our intent to never make it necessary.
