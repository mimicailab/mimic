# @mimicai/adapter-mongodb

MongoDB database adapter for [Mimic](https://github.com/mimicailab/mimic) — seeds collections with embedded documents and persona-consistent data.

## Install

```bash
npm install @mimicai/adapter-mongodb
```

## Usage

```json
{
  "databases": [
    {
      "adapter": "mongodb",
      "connectionString": "mongodb://localhost:27017/testdb"
    }
  ]
}
```
