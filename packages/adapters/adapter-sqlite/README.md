# @mimicai/adapter-sqlite

SQLite database adapter for [Mimic](https://github.com/mimicailab/mimic) — seeds tables with persona-consistent data using better-sqlite3.

## Install

```bash
npm install @mimicai/adapter-sqlite
```

## Usage

```json
{
  "databases": [
    {
      "adapter": "sqlite",
      "connectionString": "./data/test.db"
    }
  ]
}
```
